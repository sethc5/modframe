# Naming Conventions

## Sections

Section folders live under `topics/` and are named:

- `NN_<Section_Slug>`

Example: `11_Accountability_and_Transparency_Gaps`

## Topics

Topic folders live under a section and are named:

- `NNN_<Topic_Slug>`

Example: `167_Inspectors_General_Reporting_Structures`

## Files inside a topic

- `README.md`
- `outline.md`
- optional `figures/`
