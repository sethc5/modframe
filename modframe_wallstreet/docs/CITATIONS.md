# Citations

Aim: make claims traceable.

## Minimum citation fields

- Title
- Issuing body / author
- Date
- URL (or archive link)
- Relevant section/page if possible

## Preferred sources

- SEC rules and releases, CFTC regulations, OCC guidance, FRB orders and data
- FDIC reports, FINRA rules, court opinions
- EDGAR filings (10-K, 10-Q, 8-K, proxy statements, Form ADV, 13F)
- Official datasets (TRACE, FRED, Call Reports, FOCUS reports)
- GAO / OIG reports on financial oversight
